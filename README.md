Tresenzettel App
================

Android App zum zählen von Kassen- und Getränkebeständen bei Veranstaltungen.

Terminologie
------------

* sheets = Tresenzettel
* stock = Bestand
* opening stock = Anfangsbestand
* final stock = Endbestand
* beverages = Getränke
* cases = Kästen
* bottles = Flaschen
* opening balance = Kasse vorher
* final balance = Kasse nachher
* revenue = Umsatz
